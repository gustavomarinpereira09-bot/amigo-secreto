# 🎁 Amigo Secreto

Projeto desenvolvido durante os cursos de Lógica de Programação e JavaScript da Alura.

## 📖 Sobre o projeto

O Amigo Secreto é uma aplicação web que permite cadastrar participantes em uma lista e realizar um sorteio aleatório entre eles.

Este projeto foi desenvolvido como parte das atividades práticas da plataforma Alura, com o objetivo de aplicar conceitos fundamentais de programação e desenvolvimento web.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎮 Como utilizar

1. Digite o nome de um participante.
2. Clique em **Adicionar** para incluí-lo na lista.
3. Adicione todos os participantes desejados.
4. Clique em **Sortear amigo** para realizar o sorteio.
